# Launch Checklist

## Pre-Launch

- Production environment configured.
- Database migrations tested.
- Backups configured.
- Object storage configured.
- Email/SMS/payment credentials configured.
- Domain and SSL configured.
- Admin account created.
- Staff roles created.
- Seed data reviewed.

## Training

- Admin training.
- Reception training.
- Operations manager training.
- Caretaker/housekeeping training.
- Restaurant/kitchen training.
- Finance/reporting training.

## Launch Day

- Freeze demo/test data.
- Deploy production build.
- Smoke test login, booking, payment, check-in, housekeeping, reports.
- Monitor logs and errors.
- Confirm backup job.

## Post-Launch

- Track issues daily for first week.
- Review staff feedback.
- Tune reports and workflows.
- Plan v1.1 improvements.
