# UX Design Template

## Experience Principles

- Guest flows must feel simple, trustworthy, and fast.
- Admin flows must be dense, scannable, and operational.
- Mobile staff flows must prioritize large tap targets and quick status updates.
- Dark and light mode must both feel intentional.

## Required Screens

### Guest

- Room catalog
- Room detail
- Booking form
- Booking confirmation
- Booking lookup

### Staff

- Login
- Admin dashboard
- Front desk calendar
- Booking detail
- Guest profile
- Check-in
- Check-out
- Operations manager dashboard
- Housekeeping task queue
- Maintenance/damage report
- Asset verification
- Restaurant orders
- Kitchen display
- Reports overview

## Screen Spec Format

For each screen define:

- User role
- Goal
- Primary actions
- Data shown
- Empty state
- Loading state
- Error state
- Offline behavior
- Permissions
- Mobile layout notes
- Desktop layout notes
